# Homicide Analysis (Washington Post Dataset)

## Περιγραφή
Ανάλυση δεδομένων ανθρωποκτονιών σε μεγάλες πόλεις των ΗΠΑ (2007–2017) με Python/Jupyter.
Περιλαμβάνει καθαρισμό δεδομένων, exploratory analysis και οπτικοποιήσεις (χρονικές τάσεις, δημογραφικά θυμάτων, χάρτες).

## Αρχεία
- `homicideanalysis.ipynb`: Jupyter Notebook ανάλυσης
- `data/homicide-data.csv`: Dataset
- `figures/`: Screenshots/διαγράμματα (προαιρετικά)
- `report/`: Τελικό report (προαιρετικά)

## Πώς να τρέξει
1. `pip install -r requirements.txt`
2. Άνοιξε το notebook και τρέξε τα cells.

## Πηγή δεδομένων
Washington Post – Unsolved Homicide Database (2007–2017).
